## About This Repo

This repo contains an experimental **autonomous AI agent** built with Python, Docker, and multiple local LLMs.  
The system orchestrates **5 local LLMs** sequentially for safe memory usage:

| Role | Model | Purpose |
|------|-------|---------|
| Architect | Llama 3.1 8B | Breaks tasks into step-by-step plans |
| Lead Developer | Qwen 2.5-Coder 14B | Generates core code |
| Debugger | DeepSeek-Coder 6.7B | Optimizes/fixes code |
| Quick Edits | Qwen 2.5-Coder 7B | Fast modifications |
| Documentation | Llama 3.2 | Explains code & generates docs |
